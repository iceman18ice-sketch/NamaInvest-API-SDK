## 🛡️ بروتوكولات الأمان العسكرية في نما إنفست (Military-Grade Protocols)
كشركة تعتمد على (نما إنفست) لإدارة أصولها المالية، رواتب موظفيها، وضريبتها مع الدولة (ZATCA)، الأمان ليس رفاهية؛ بل هو الجوهر. بينما تعاني الأنظمة مفتوحة المصدر (כمثل Odoo) من ثغرات الحقن المستمرة (SQL Injection) والتسريبات إذا لم يتم إغلاقها من مهندس مختص، يوفر (نما إنفست) حماية سحابية غير قابلة للاختراق (Zero-Trust).

### 1. المصادقة والتحكم بالوصول (NextAuth & JWT)
- **بروتوكول JSON Web Tokens (JWT):** يتم تشفير جلسات تسجيل الدخول للموظفين والكاشير برموز ديناميكية تتغير باستمرار.
- **صلاحيات مجهرية (Granular RBAC):** الإدارة تحدد بدقة متناهية ما يمكن للكاشير رؤيته. المحاسب يرى القيود ولا يرى الرواتب؛ الكاشير يرى نقطة البيع لفرعه فقط ولا يرى مستودع فرع آخر.

### 2. التشفير السحابي وقواعد البيانات (PostgreSQL RLS)
- **Level Security (RLS) Row:** حتى لو تم اختراقชั้น الـ API الخارجي (وهو مستحيل مع جدران Vercel Edge Firewall)، فإن قاعدة البيانات تمنع المستخدم (Tenant A) من قراءة أي صف (Row) يخص (Tenant B) برمجياً ومن الجذور (Hardcoded Isolation).
- **SSL / TLS 1.3:** جميع اتصالات المزامنة مع متجرك في (سلة أو زد) ومع هيئة الزكاة تتم عبر أنفاق مشفرة 100%.

### 3. الحماية ضد هجمات الإغراق (DDoS & Rate Limiting)
خوادم (نما إنفست) محمية عبر درع (Edge Network) عالمي يوزع الحمل ويمتص أي هجوم (DDoS). لن تتوقف مبيعاتك في (البلاك فرايداي) مهما كان الضغط والزحام على السيرفرات.
